# Aether Link (以太链接)
### —— 工业复杂系统的概率因果推演与可视化平台

**Aether Link** 不仅仅是一个数据可视化工具，它是一个基于**贝叶斯网络 (Bayesian Network)** 的动态仿真系统。它旨在帮助工程师和管理者在复杂的工业 BOM（物料清单）结构中，通过科学的“假设-观测-推演”闭环，洞察系统的脆弱性，模拟故障传播，并验证设计方案的鲁棒性。

---

### 1. 核心理念：科学的推演逻辑
Aether Link 的设计严格遵循科学实验的方法论，将“理论计算”与“现实观测”解耦，确保推演过程的严谨性。

*   **先验 (Priors) 与 属性 (Attributes)：** 用户通过定义节点的 Tag（如质量等级、工艺精度）来设定系统的**理论状态**。这是推演的基准。
*   **观测 (Evidence) 与 铁证 (Hard Fact)：** 支持“实验模式”。用户可以手动锁定任意节点的置信度（如强制设定某零件损坏）。系统将其视为**既定事实**，优先级高于一切理论计算，从而阻断上游干扰，建立稳定的实验边界。
*   **动态推演 (Dynamic Inference)：** 只有当“观测”被撤销（重置为计算值）时，贝叶斯引擎才会重新接管，依据上下游的因果关系和节点的内部属性，自动计算出全网络的概率分布。

### 2. 视觉体验：Neon Neural Network (霓虹神经网络)
我们重新定义了数据可视化的美学，将枯燥的拓扑结构转化为具有生命力的数字有机体。

*   **Quantum Lock (量子锁定) 架构：** 解决了传统 3D 拓扑图在数据更新时“爆炸/重排”的顽疾。无论数据如何剧烈跳动，网络结构始终保持岩石般稳定，仅在视觉层进行平滑的属性突变。
*   **Pulse Engine (脉冲引擎)：** 数据的变化不再是冷冰冰的数字跳动，而是通过**生物发光 (Bioluminescence)** 般的脉冲表现出来。当节点健康度发生显著变化时，会触发一道耀眼的能量脉冲，直观地展示出“故障”或“恢复”在网络中的传播路径。
*   **多维视图切换：**
    *   **Topology Mode (拓扑维)：** 强调结构归属，以 EBOM/MBOM/SBOM 颜色区分，展示“骨架”。
    *   **Health Mode (健康维)：** 强调状态监测，以红/琥珀/绿三色热力图展示“生命体征”。
*   **Moonlit Web (月光织网) 风格**：未激活的节点隐退为银白色的幽灵结构，激活的链路则如霓虹般闪耀。这种极致的明暗对比（Extreme Contrast）让用户的注意力始终聚焦在关键路径上。

### 3. 系统能力：全链路仿真
*   **多源算法融合：** 内置多种概率算法（BN 贝叶斯网络, MCMC 马尔可夫链蒙特卡洛, MCS 蒙特卡洛模拟），可根据数据规模和精度要求灵活切换。
*   **全 BOM 覆盖：** 支持从 EBOM (设计) 到 MBOM (制造) 再到 SBOM (服务) 的全生命周期数据映射，打通“设计-制造-运维”的数据孤岛。
*   **智能辅助：** 集成 AI Assistant，能够理解当前的图谱结构，辅助用户进行复杂的查询和操作。

### 4. 应用场景
*   **故障根因分析 (RCA)：** 当后端出现故障时，通过反向推演，快速锁定可能性最高的前端设计缺陷或工艺偏差。
*   **设计鲁棒性验证：** 在设计阶段“注入”虚拟故障（Inject Fault），观察其对整个系统的冲击范围，从而优化冗余设计。
*   **供应链风险评估：** 模拟某一特定供应商（节点）断供或质量波动对整体交付置信度的影响。
